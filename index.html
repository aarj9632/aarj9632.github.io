  • You can't put anything after the text because there is no free space available.

  • For example :

  • Lets talk about the attributes "id" and "class" :

  • ID :

  • The id of an element is like the name of a person. Just like everyone has a name in the same way you can give any element a name which is called id in HTML.
  • After assigning id to an element you can use it's id to style, script or link the element in CSS or Javascript.
  • Also you can use one id for any one element and you can't use the same id for any other element.
  • If you want to know more about ID attribute then Click Here.

  • Class :

  • As the name suggests you can put multiple elements into the class attribute and access, style or script all the elements through CSS or Javascript.
  • If you want to know more about class attribute then Click Here.
